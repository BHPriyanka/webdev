Secure Instant Messaging System

SIMS is a single Server multi Client architecture based system developed in python.
Protocols:

Key establishment Protocol for Authentication with Peers
Messaging Protocol for Communication
Logout Protocol to Disconnect
Security features:
PFS, End-Point Hiding, Protection against use of weak passwords
Resistant to:
Replay attack, Reflection attack, Impersonation and Passive Man-In-The-Middle attack
